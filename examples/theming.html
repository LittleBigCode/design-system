Theming

Themes override only the semantic (Tier 2) tokens — primitives and component rules are untouched, so a full re-skin is ~20 lines. Switch the theme below and watch every component (and this page's chrome) re-skin live.

Switch theme

This sets data-theme on the document root. Dark ships in css/themes/dark.css; Sepia is a worked brand example in css/themes/sepia.css (built only from the brand palette). See docs/theming.md.
