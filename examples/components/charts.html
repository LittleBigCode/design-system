Components

Charts

A small set of pure-SVG charts — line, area, donut, pie, gauge and stacked bar — with no chart library. Each shares the flat .ds-chart wrapper, faint .ds-chart__grid gridlines, .ds-chart__axis labels and a wrap-friendly .ds-chart-legend. Series colors come from the categorical --ds-chart-1…--ds-chart-6 token palette. Flat, sharp, no radius, no shadow.

Line

A .ds-linechart draws one __line polyline per series across a shared y-range, over optional gridlines and x-axis labels, with a dot on every point.

Gauge

A .ds-gauge draws a faint 270° __track arc behind a colored __value arc (length via stroke-dasharray), with a big __center number and an optional uppercase __label. The arc color defaults to --ds-chart-1; pass a color or value→color thresholds.

Single value · center number · label

Stacked bar

A .ds-stackedbar lays each datum out as a __row: an optional __label and a 100%-wide __track of proportional __seg segments (width + color set inline, with a title hover tooltip). A shared .ds-chart-legend names the segments.

Bar

A .ds-barchart lays each datum out as a __col: an optional __label, a __track with a proportional __bar (length is value/max), and a __value. The default orientation is vertical; horizontal flips it to rows. A status recolors a bar (.is-success, .is-danger, …).
