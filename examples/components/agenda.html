Components

Agenda

A chronological scheduler: events grouped under per-day headers (uppercase faint weekday + date on a top rule), each row pairing a tabular time column, a small square status marker, and a body with a title and optional meta. Rows shift to bg-alt on hover. Flat & sharp — 1px rules, no radius, no shadow.

Grouped events

Events are sorted and bucketed by day; each group opens with a date header. The marker recolors by status via is-*.

React

The <Agenda> component sorts and groups an events array by day; status maps to the marker's is-* class, and an emptyMessage covers the empty case.

<Agenda> 
import { Agenda } from "@littlebigcode/design-system/react";

<Agenda
  emptyMessage="Nothing scheduled."
  events={[
    { date: "2026-06-18", time: "09:00", title: "Publish rate matrix", meta: "All rates synced.", status: "success" },
    { date: "2026-06-18", time: "11:30", title: "1:1 with the Director", status: "info" },
    { date: "2026-06-18", time: "16:00", title: "Quarter deadline", status: "danger" },
    { date: "2026-06-19", time: "10:00", title: "Review draft", meta: "12 rows missing a currency.", status: "warning" },
    { date: "2026-06-19", time: "14:00", title: "Office hours" },
  ]}
/>
