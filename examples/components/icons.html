Components · foundations

Icons

A Lucide-compatible line-icon set: every glyph is drawn in a 24×24 box with fill="none", stroke="currentColor" and a 2px round stroke, so icons inherit text color and any Lucide icon can be dropped in unchanged. Use the React <Icon>, the <ds-icon> web component, or paste the raw <svg>.

Library

All built-in icons at 24px. Sized via font-size or width/height; color follows currentColor.

.ds-icon

Sizing & color

Icons are 1em square by default, so they scale with font-size and inherit currentColor. Set width/height (or the React size prop) to fix a size.

React

<Icon>
import { Icon } from "@diametral/design-system/react";

// Decorative — inherits font-size and currentColor
<Icon name="search" />

// Inside a button, with a fixed size
<button className="ds-button">
  <Icon name="download" size={18} /> Export
</button>

// Meaningful icon — give it an accessible name
<Icon name="bell" title="Notifications" size={24} />

Web component

<ds-icon>
<script type="module" src="components/ds-icon.js"></script>

<ds-icon name="search"></ds-icon>
<ds-icon name="download" size="18"></ds-icon>
<ds-icon name="bell" size="24" title="Notifications"></ds-icon>

Raw SVG

.ds-icon markup
<svg class="ds-icon" width="24" height="24" viewBox="0 0 24 24"
     fill="none" stroke="currentColor" stroke-width="2"
     stroke-linecap="round" stroke-linejoin="round" aria-hidden="true">
  <circle cx="11" cy="11" r="8"></circle>
  <path d="m21 21-4.3-4.3"></path>
</svg>
