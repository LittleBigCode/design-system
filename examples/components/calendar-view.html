Components

Calendar

A full-size month view on a flat bordered surface: a head with the month label, prev / next icon buttons and a Today action, a faint uppercase weekday row, then a 7-column grid of equal cells separated by 1px rules. Each day shows its date top-left and stacks small status-colored event chips, capped with a "+N more" line. Distinct from the date picker's compact popover.

Month view

A live month with sample events. Use the prev / next buttons to navigate; today gets an accent ring and the selected day fills with the accent.

React

The <Calendar> component builds the month grid from plain-JS date math and places events in day cells. The shown month is controlled via month + onMonthChange, or managed internally.

<Calendar> 
import { Calendar } from "@littlebigcode/design-system/react";

<Calendar
  month="2026-06-01"
  value={selected}
  weekStartsOn={1}
  onSelectDate={(date, iso) => setSelected(iso)}
  onMonthChange={(month) => console.log(month)}
  events={[
    { date: "2026-06-03", label: "Publish rates", status: "success" },
    { date: "2026-06-12", label: "Review draft", status: "warning" },
    { date: "2026-06-18", label: "Quarter deadline", status: "danger" },
    { date: "2026-06-18", label: "Sync call", status: "info" },
  ]}
/>
